# プライバシーポリシー（Privacy Policy）

最終更新日: 2026年6月26日

> 本書は一般的なひな型です。法的有効性を保証するものではなく、必要に応じて弁護士等の専門家にご相談ください。

本プライバシーポリシーは、本アプリ「Career Roadmap」（以下「本アプリ」）における利用者の情報の取り扱いについて、本アプリの開発者（以下「当方」）が定めるものです。

## 1. 基本方針

本アプリは、**サーバーを持たず、利用者の端末内で完結**して動作します。当方は、利用者が本アプリに入力・保存したデータ（日課・履歴・目標・APIキー等）を**収集・送信・閲覧・保存することはありません**。

## 2. 端末内に保存されるデータ

本アプリのすべてのデータは、利用者の端末のブラウザの **localStorage** にのみ保存されます。これらのデータは利用者の端末内にとどまり、当方を含む第三者に自動的に送信されることはありません。

## 3. 第三者サービスへのデータ送信（AIコーチ機能）

1. 利用者が AIコーチ機能を利用する場合に限り、本アプリは利用者の入力内容や進捗の要約を、利用者自身が設定した API キーを用いて **Google Gemini API** に送信します。
2. 送信されたデータの取り扱いは、Google のプライバシーポリシーおよび利用規約に従います。当方はこれらのデータの取り扱いについて責任を負いません。
3. AIコーチ機能を利用しない場合、外部へのデータ送信は行われません。

## 4. API キーの取り扱い

利用者が設定した Gemini API キーは、利用者の端末内（localStorage）にのみ保存され、当方に送信されることはありません。API キーの管理は利用者の責任で行ってください。

## 5. Cookie・アクセス解析

本アプリは、利用者を追跡する目的での Cookie やアクセス解析ツールを使用しません。

## 6. お問い合わせ

本ポリシーに関するお問い合わせは、本アプリのリポジトリの Issues 等を通じてお願いします。

## 7. 本ポリシーの変更

当方は、必要に応じて本ポリシーを変更することがあります。変更後の本ポリシーは、本アプリまたはリポジトリ上に掲示した時点で効力を生じます。
